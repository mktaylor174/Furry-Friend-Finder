# Furry-Friend-Finder
Developed by Maggie Taylor

This contains the demonstration video for Furry Friend Finder, an AI agent I designed to help match users with the best potential pet for them.

I created this agent to help solve the issues surrounding owners who adopt pets that turn out to not be great fits for them, or to help those unsure of what to adopt next figure out their next pet. No matter the initial situation, Furry Friend Finder could take your concerns and desires, and use those to match you with the species and breed of pet that best suits you.

This was created using NeuralSeek's mAIstro AI agent builder. I then had to apply my knowledge of prompt engineering, ethical data collection, and data analysis to ensure the agent worked properly. The agent featured the collection of user preferences, matching to a database of various pet breeds and species, and personalized recommendations.

The creation of this agent taught me many valuable lessons in agentic AI development, data collection and refinement, and proper prompt engineering.
